# Screenshare-Alliance
Script desarrollado por Screenshare Alliance con el objetivo de facilitar y acelerar las Screenshares (SS).
Incluye múltiples scripts automatizados y acceso rápido a tools utilizadas por la comunidad de SS, permitiendo una revisión más eficiente y ordenada.
También ofrece descargas de Tools de desarrolladores reconocidos como Spokwn, Orbdiff, entre otros, centralizando todo en un solo lugar para optimizar el proceso de detección.
